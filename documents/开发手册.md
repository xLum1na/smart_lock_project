# 开发手册

## 一、系统架构设计流程

### 1、明确核心场景与行为流

#### (1)主流程

* 启动流程
  * 上电/复位
  * 硬件初始化
  * 加载系统配置
  * 进入睡眠

* 人脸验证流程
  * 触发人体靠近中断
  * 设备上电
  * 人脸获取与检测
  * 验证成功/失败
  * 进入睡眠

* 密码验证流程
  * 触发人体靠近中断
  * 硬件上电
  * 触发lcd触摸
  * 进入密码输入界面
  * 验证成功/失败
  * 进入睡眠

* 指纹验证流程
  * 触发人体靠近中断
  * 硬件上电
  * 触发指纹模块指纹靠近中断
  * 进入指纹验证界面
  * 验证成功/失败
  * 进入睡眠

* 验证通过流程
  * 验证成功
  * 显示验证成功提示
  * 开锁
  * 进入睡眠

* 验证失败流程
  * 验证失败
  * 显示验证失败提示
  * 重试
  * 进入睡眠

#### (2)异常处理流程

* 连续试错流程
  * 记录失败累计次数，若连续5次错误（任意方式组合），触发锁定。
  * 锁定期间拒绝所有开锁尝试，LCD提示“已锁定，请等待X分钟”。
  * 启动定时器，锁定持续时间1~3分钟（可配置，默认1分钟）。
  * 定结束后自动清零失败计数，恢复正常待机。
  * 每次锁定事件写入报警记录表。

* 电池电量低流程
  * 每次上电或开锁操作后检测电池电压。
  * 若剩余电量 ≤ 10%，触发低电量报警。
  * LCD提示“电量低，请及时更换电池”
  * 每次成功开锁后再次重复提示
  * 电量 ≤ 3% 时，仅保留机械钥匙开锁能力，电子系统拒绝电机驱动（但可导出日志）。
  * 低电量事件写入报警记录表。

* 加载系统配置异常
  * 启动时读取配置分区（用户表、设备信息等）并进行 CRC 校验。
  * 若校验失败或配置损坏：
    * 使用出厂默认配置（无用户，管理员密码恢复为默认“123456”或物理按键组合）。
    * LCD显示系统配置异常

  * 记录错误事件到内部错误日志（仅调试接口可读）。


* 硬件故障流程
  * LCD故障
    * 自动复位模块3次。
    * 若仍失败,仅保留机械钥匙开锁能力
  * CAMERA故障
    * 自动复位模块3次。
    * 若仍失败,提示“XX识别模块故障，请使用其他方式”。
  * 指纹故障
    * 自动复位模块3次。
    * 若仍失败,提示“XX识别模块故障，请使用其他方式”。
  * sdcord故障
    * 重试3次，仍失败则放弃操作，提示“记录存储失败”。
    * 系统继续运行，但后续记录可能丢失。
  * 舵机故障
    * 自动复位模块3次。
    * 若仍失败,仅保留机械钥匙开锁能力。
  * HC-SR501故障
    * 自动复位模块3次。
    * 若仍失败,仅保留机械钥匙开锁能力。

* 管理员操作流程
  * 输入管理员密码Ship it
  * 验证通过后进入管理员菜单：
    * 添加/修改/删除普通用Ship it户（人脸/指纹/密码）
    * 修改管理员密码
    * 恢复出厂设置
    * 导出本地记录
  * 菜单操作超时（30秒无按键）自动退出。
  * 所有配置修改立即写入 Flash，并生成操作记录日志。

* 管理人脸流程
  * 管理员菜单选择“添加人脸” → 提示普通用户放置人脸 → 3D 活体检测 → 采集特征并加密存储 → 分配 UserID → 成功/失败提示。
  * 管理员菜单选择“删除人脸” → 选择用户（按 ID 或序号） → 确认删除。

* 管理密码流程
  * 管理员菜单选择“修改密码“→ 输入新密码（6-8位） → 二次确认 → 存储 Hash+salt。


* 管理指纹流程
  * 管理员菜单选择”添加指纹“→ 提示按压指纹 3 次 → 合成模板 → 分配UserId → 加密存储。
  * 删除指纹：选择用户 → 清除指纹模板。

* 管理日志流程
  * 管理员菜单选择“导出日志”。
  * 提示“请使用 USB 连接管理终端”。
  * 检测 USB 连接（需与主机建立串口或 MSC 模式）。
  * 管理员密码二次确认。
  * 将开锁记录表、报警事件表导出为 CSV 格式。
  * 导出完成后 LED 绿灯闪烁，提示“导出完成”。


#### (4)**敏感操作分类**

- **实时性敏感**：人脸识别（≤0.5s）、开锁响应（≤1.5s）
- **功耗敏感**：待机模式（≤50μA）、人脸模块休眠（≤10μA）
- **可靠性敏感**：密码校验、电机驱动、异常恢复

### 2、提取硬约束

| 约束类别         | 具体要求                                                     |
| ---------------- | ------------------------------------------------------------ |
| **时序约束**     | - 人脸识别 ≤ 0.5s - 从识别完成至电机转动 ≤ 1.5s - 连续试错锁定时间 1~3 min |
| **功耗约束**     | - 待机平均功耗 ≤ 50μA - 识别模块工作时 ≤ 200mA，休眠 ≤ 10μA - 4 节 AA 电池续航 ≥ 8 个月（10 次/天） |
| **存储约束**     | - 本地 Flash 容量 ≥ 4MB - 开锁记录 ≥ 500 条（循环覆盖） - 用户表支持至少 20 个普通用户 + 1 个管理员 |
| **安全约束**     | - 生物特征 / 密码采用 AES-128 或 SM4 加密存储 - 密码加盐 Hash（PBKDF2） - 连续 5 次错误锁定 - C 级锁芯，防技术开启 ≥10min |
| **环境约束**     | - 工作温度 -25℃ ~ +70℃ - 工作湿度 20% ~ 93% RH - 室内 IP52，半户外 IP54 - 接触放电 ±8kV，空气放电 ±15kV |
| **机械约束**     | - 电机扭矩 ≥ 2.5 N·m - 电机寿命 ≥ 20 万次 - 适用门厚 35~55mm，可扩展至 70mm |
| **成本约束**     | - BOM 成本 300~600 元人民币（不含锁体）                      |
| **法规约束**     | - 符合 GA 374-2019《电子防盗锁》 - 符合《个人信息保护法》（生物特征本地处理） - 取得 CCC 认证 |
| **功能排除约束** | - 无任何无线通信（蓝牙 / Wi-Fi / Zigbee） - 无远程开锁、临时密码、时段权限 - 无云端备份 |

#### (3) 事件流模型

| 任务/模块              | 触发方式                    | 优先级 | 执行频率/延迟约束      | 说明                                    |
| ---------------------- | --------------------------- | ------ | ---------------------- | --------------------------------------- |
| **系统初始化任务**     | 上电/复位                   | 最高   | 一次性                 | 时钟、GPIO、外设、Flash 校验、加载配置  |
| **睡眠管理任务**       | 空闲超时（30s 无交互）      | 低     | 进入睡眠前调用         | 关闭外设电源，进入 STOP 模式            |
| **传感器轮询/中断**    | 外部中断（红外/触摸）       | 高     | 实时响应               | 唤醒系统，置位事件标志                  |
| **人脸识别任务**       | 事件标志                    | 高     | 完成 ≤0.5s             | 驱动摄像头，调用 NPU 推理，返回比对结果 |
| **指纹识别任务**       | 指纹模块中断                | 高     | 完成 ≤0.3s             | 读取指纹图像，比对模板                  |
| **密码认证任务**       | 键盘中断 + 定时器去抖动     | 中     | 按键响应 ≤50ms         | 处理虚位密码，哈希比对                  |
| **电机控制任务**       | 认证成功事件                | 高     | 开锁 ≤1.5s             | 驱动电机，检测堵转                      |
| **记录存储任务**       | 事件队列（开锁/报警）       | 中     | 非实时，可缓冲         | 写入 Flash（磨损均衡），循环覆盖        |
| **低电量监测任务**     | 定时器（每 1 小时或开锁后） | 低     | 实时性无要求           | ADC 采样电压，触发报警                  |
| **失败计数与锁定任务** | 认证失败事件                | 高     | 立即处理               | 累加计数器，达到阈值切换状态            |
| **管理员菜单任务**     | 设置键长按事件              | 中     | 交互式，超时自动退出   | 驱动 LCD、键盘，执行用户管理、导出等    |
| **USB 通信任务**       | USB 插入中断                | 低     | 仅在导出日志时启用     | 实现自定义 CDC 或 MSC，传输 CSV 文件    |
| **看门狗喂狗任务**     | 定时器（周期 1s）           | 最低   | 必须在看门狗超时前喂狗 | 保证系统不死锁                          |
