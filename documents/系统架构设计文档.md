## 二、硬件描述

### 1、MCU选型

**核心处理器**

| 参数         | 规格                                |
| ------------ | ----------------------------------- |
| **CPU**      | Xtensa® 32位 LX7 双核处理器         |
| **主频**     | 最高 240 MHz（基于 40MHz 晶振倍频） |
| **浮点运算** | 支持单精度浮点运算单元（FPU）       |
| **ROM**      | 384 KB（只读，用于启动和内核功能）  |
| **SRAM**     | 512 KB（静态随机存储器）            |
| **TC SRAM**  | 16 KB                               |

**存储配置详解**

| 存储类型  | 容量   | 接口类型         | 说明                               |
| --------- | ------ | ---------------- | ---------------------------------- |
| **Flash** | 16 MB  | Quad SPI（QIO）  | 程序固件存储，最大支持 120MHz 时钟 |
| **PSRAM** | 8 MB   | Octal SPI（OPI） | 伪静态随机存储器，扩展数据内存     |
| **SRAM**  | 512 KB | 片上             | 内核运行时内存                     |

**无线通信**

- **Wi-Fi**：2.4 GHz，802.11 b/g/n，802.11n 模式下最高 150 Mbps，支持帧聚合（A-MPDU/A-MSDU）
- **蓝牙**：低功耗蓝牙（Bluetooth LE），支持 Bluetooth 5 和 Bluetooth Mesh
- **速率**：125 Kbps、500 Kbps、1 Mbps、2 Mbps
- **共存**：Wi-Fi 与蓝牙共用同一根天线，支持共存机制

**外设接口**

ESP32-S3-N16R8 拥有丰富的外设接口，包括：

- **通用接口**：GPIO（最多 45 个可编程）、UART、I2C、I2S、SPI、PWM
- **多媒体**：LCD 接口、Camera 接口
- **USB**：全速 USB 2.0 OTG、USB Serial/JTAG 控制器
- **其他**：ADC、触摸传感器、温度传感器、红外遥控、脉冲计数器、LEDPWM、MCPWM、SDIO 主机接口、GDMA、TWAI 控制器（兼容 CAN ISO11898-1）

**电器参数**

| 参数           | 值                                                         |
| -------------- | ---------------------------------------------------------- |
| **工作电压**   | 3.0 ~ 3.6 V                                                |
| **输入电压**   | 5 V（通过 USB）                                            |
| **发射电流**   | 285 ~ 355 mA                                               |
| **接收电流**   | 95 ~ 97 mA                                                 |
| **发射功率**   | 20.5 dBm                                                   |
| **接收灵敏度** | -103.5 dBm                                                 |
| **工作温度**   | -40℃ ~ +65℃（标准版）；另有 -40~85℃、-40~105℃ 扩展温度版本 |
| **封装**       | SMD，25.5 × 18 mm（WROOM-1）/ 18 × 19.2 mm（WROOM-1U）     |
